1. 用法示例

<button class="btn btn-primary" id="purebox">purebox</button>
简单调用 
/*jQuery工具集调用方式*/
​$('#purebox').click(function() {
	$.pb({
		id : 'purebox',
		title : '标题',
		content : '好雨知时节， 当春乃发生。随风潜入夜， 润物细无声。'
	});
});

/*jQuery包装集调用方式*/
​$('#purebox').pb({
	id : 'purebox',
	title : '标题',
	content : '好雨知时节， 当春乃发生。随风潜入夜， 润物细无声。'
});

2. 配置参数

名称 类型 默认值 描述
title String '标题' 标题内容。
content String
HTMLElement
jQeury Obj		 '' 消息内容。
如果传入的是HTMLElement类型，如果是隐藏元素会给其设置display:block以显示该元素，其他属性与绑定的事件都会完整保留，对话框关闭后此元素又将恢复原来的display属性，并且重新插入原文档所在位置。
width Number 'auto' 设置弹出层内容区宽度。
height Number 'auto' 设置弹出层内容区高度。
top Number 'c' 相对于可视区域的X轴的坐标。默认居中。
left Number 'c' 相对于可视区域的Y轴的坐标。默认居中。
fixed Boolean true 是否开启静止定位。
drag Boolean true 是否允许拖拽。
mask Boolean true 是否开启遮罩。
resize Boolean false 是否允许调节尺寸。需要引入jquery.resizable.js。
head Boolean true 是否显示标题栏。
foot Boolean true 是否显示按钮栏。
xBtn Boolean true 是否显示关闭按钮。
cBtn Boolean true 是否显示取消按钮。
onClose Function null 关闭按钮回调函数。
onOk Function null 确定按钮回调函数。
onCancel Function null 取消按钮回调函数。

3. 扩展方法

名称 参数 描述
resize() width, height 使用已实例的弹出层对象，重新设定其大小。
setPos() top, left 使用已实例的弹出层对象，重新设置位置
center() 设置居中。
setContent() String
HTMLElement
jQeury Obj		 设置弹出层内容区高度。
dispose() String
HTMLElement
jQeury Obj		 销毁对话框，如果传入的是HTMLElement或jQeury类型，对话框销毁后此元素将重新插入原文档所在位置。
